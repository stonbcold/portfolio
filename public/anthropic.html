Mon projet

Objectif
Construire un agent autonome capable de recevoir une tâche en langage naturel, de la décomposer, d'appeler les bons outils, puis de restituer un résultat structuré.

  • ✓

Apprentissages

Ce que j'ai retenu

  • 01
    Prompting avancé L'efficacité d'un agent dépend autant de la qualité du system prompt que du code Python qui l'entoure.
  • 02
    Tool Use en profondeur Définir un bon schéma JSON pour ses outils est critique — ambiguïté = mauvaise décision du modèle.
  • 03
    Sécurité des clés API Gestion propre des variables d'environnement (.env), jamais de clé en dur dans le code versionné.
  • 04
    Gestion des coûts Optimiser les appels API (tokens, cache prompt) est une compétence à part entière en production.
